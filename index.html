关于我

探索者，构建者，学习者

我是一名热爱技术的探索者，专注于 AI Agent 生态和多智能体协作系统。 从大语言模型的 Prompt Engineering 到复杂的 Multi-Agent 编排架构， 我享受将前沿 AI 能力转化为实际可用的工具和工作流。

我相信技术的价值在于解决真实问题。无论是自动化日常繁琐任务， 还是构建智能助手提升团队效率，我始终追求「能用 → 好用 → 易用」的递进。 目前正在探索 AI 驱动的个人知识管理系统和 Vibe Coding 最佳实践。

工作之余，我喜欢写技术博客、在社区分享经验， 并通过开源项目连接志同道合的伙伴。
